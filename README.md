💼 Personal Portfolio Website
📌 Project Overview

This project is a personal portfolio website developed as part of Task 1 for the Full Stack Web Development Internship at Future Interns.

The portfolio highlights my skills, projects, resume, and contact information, providing a centralized platform to showcase my development work and professional profile.

The website is designed with a modern UI, responsive layout, and smooth navigation to ensure a seamless user experience across different devices.

🚀 Features

Modern hero section introducing my profile

Detailed About Me section describing my background and goals

Skills section showcasing technical abilities

Projects section displaying my work and development experience

Resume section with downloadable CV

Contact form for communication

Responsive design compatible with desktop, tablet, and mobile devices

🛠 Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and responsive design

JavaScript – Interactivity and dynamic behavior
